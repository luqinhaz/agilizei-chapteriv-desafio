# Instalação
Instalar o [Node.js](https://nodejs.org/en/download/ "Node.js")
Versões utilizadas:
- Node version: v14.16.0
- NPM version: 6.14.11

Clonar o projeto:
`git clone LINKDOPROJETO.git`

Navegar para a pasta do projeto e instalar as dependencias:
`cd desafio02Chapteriv`
`npm install`
